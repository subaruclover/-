欢迎来到实战云多智能体强化学习项目大纲
======================================

.. image:: /_static/images/honor-of-kings.png
   :width: 600
   :alt: Honor Of Kings

实战云多智能体强化学习项目旨在帮助更多的高校，能够应用开悟平台开展人工智能教学工作，本项目将面向中国本科人工智能专业领域，设计一套完整的课程方案。

* 第一次在线meetup会议: 2022年10月11日, 14:00-15:00（GMT+08:00）

  参加者: 任川, Nicole, Qiong

* 第二次在线meetup会议: 2022年10月28日, 15:00-15:30（GMT+08:00）

  参加者: 任川, 张静娴, 黄琼

查看 :doc:`usage` 和 :doc:`intro` 部分了解更多信息。
